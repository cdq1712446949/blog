举例用的表

a_table： 

CREATE TABLE `a_table` (
  `a_id` int(11) DEFAULT NULL,
  `a_name` varchar(10) DEFAULT NULL,
  `a_part` varchar(10) DEFAULT NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8

b_table： 

CREATE TABLE `b_table` (
  `b_id` int(11) DEFAULT NULL,
  `b_name` varchar(10) DEFAULT NULL,
  `b_part` varchar(10) DEFAULT NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8

测试数据：

一  内连接

语法：a_table inner join b_table

语句： 

select * from a_table a inner join b_table b on a.a_id=b.b_id

过程：内连接会根据on的条件生成临时表，符合条件的记录才会被添加到临时表

临时表：mysql内外连接时会生成一个临时表，临时表有两个表的所有字段

执行结果

二  左连接（左外连接）

语法：a_table left outer join b_table   (outer可以省略)

语句： 

select * from a_table a right join b_table b on a.a_id=b.b_id

过程：左连接会把左表（a_table）记录全部添加到临时表，右表根据on的条件添加，不符合条件的记录都为null

三 右连接（右外连接）

和左连接相反

语句： 

select * from a_table a left join b_table b on a.a_id=b.b_id
